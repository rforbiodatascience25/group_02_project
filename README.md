### Description

In this project we will analyse the data from the paper "**Apoptosis and other immune biomarkers predict influenza vaccine responsiveness**". In the original paper, researchers used blood samples from 89 healthy people before and after vaccination and sequenced them in Illumina BeadScan. We will combine the test data with the metadata given and then analyse them to visualise the response to the vaccine based on the age, gender, BMI, CMV or EBV possitive and type of antibody.
